# FinApply.ai

**Prove How You Think. Not Just What You Know.**

FinApply.ai is an AI-powered assessment platform that evaluates candidates for finance roles through realistic deal simulations and produces a **FISS (Financial Intelligence Simulation Score)** — a standardised, shareable credential.

---

## ✨ Live

🌐 **[fin-apply-ai.vercel.app](https://fin-apply-ai.vercel.app)**

---

## 🧠 What Is the FISS Score?

The Financial Intelligence Simulation Score evaluates candidates across 4 dimensions:

| Dimension | What It Measures |
|-----------|-----------------|
| **Financial Reasoning** | Accuracy of analysis, valuation logic, number sense |
| **Structured Thinking** | Framework application, logical flow, prioritisation |
| **Risk Identification** | Spotting risks, regulatory awareness, downside scenarios |
| **Decision Clarity** | Actionable recommendations, conviction, communication |

Each dimension is scored 0-25 for a total of 100 points.

---

## 🏗️ How It Works

1. **Register** — Choose your target finance role (IB, PE, Big 4, ER, CF).
2. **Simulate** — Complete a 45-minute Deal Room simulation with real company financials.
3. **Get Scored** — Receive your FISS Report with dimension breakdowns and actionable feedback.
4. **Share** — Use your FISS Score as a portable credential with employers.

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/chinmaykhatri/FinApply.Ai.git
cd FinApply.Ai

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local

# Start development server
npm run dev
```
